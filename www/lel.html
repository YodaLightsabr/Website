ThunderBot

ThunderBot is a multi-purpouse Discord bot that features an economy system, music, leveling, games, moderation tools, memes, web searching, and image generation. You can check it out at ThunderBot.cf.

DisGuard

DisGuard is a bot that allows you to password-protect invite links, and has a lot more planned in the near-future. DisGuard also has a website, at DisGuard.cf.

RobinDB

RobinDB is a key/value database built for the cloud. RobinDB has a similar interface to Quick.db, but is a wrapper of NeDB instead, which makes it asynchronous and reduces data loss. RobinDB is 50x faster than Quick.db. You can view it on GitHub here.

Frontly.js

Frontly.js lets you quickly make dynamic webpages in Node.js. Its easy to use and has a simple syntax. It's on NPM at frontly and can be found on GitHub here.
